# Trismart Intelligent Automation

Welcome to the Trismart frontend application! This project is built using React, Vite, TypeScript, and Tailwind CSS.

## Features
- **Responsive Layout**: Designed mobile-first, scales perfectly to desktop and ultra-wide screens.
- **Glassmorphism**: High-end UI features utilizing backdrop-blur, custom transparency, and sleek borders.
- **Micro-Animations**: Framer Motion powered transitions for an immersive experience.
- **Integrated Irames**: Seamless rendering of external content engines.
- **FormSubmit.co API**: Email handler configuration allowing easy contact forms without heavy backends like Firebase.

## Deploying to Vercel

This app is optimized and configured for seamless deployment to Vercel forever for free.

1. **Connect GitHub/GitLab/Bitbucket** to your Vercel account.
2. Select **"Import Project"** and choose this repository.
3. Vercel will automatically detect that this is a **Vite** project.
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Click **Deploy**.

*Note: The `vercel.json` file is already included in this repository to manage routing and strict HTML headers so you don't encounter 404 errors if pages are reloaded.*
